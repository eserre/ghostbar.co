Todoist, lista de tareas online + Plugin para Chrome

June 07, 2011

Las llamadas ToDo-lists, task-lists o en español listas de tareas, abundan en la internet, con varios enfoques, como Strikeapp (el hermano pequeño de Resolve, una aplicación web para manejar proyectos) con un enfoque a la sencillez, no-reusable y gran diseño; Tada-lists (el hermano pequeño de Basecamp) siguiendo el modelo de Basecamp; TeuxDeux con genial diseño e implementación, incluyendo pasar al día actual las tareas que quedaron por hacer y Todoist, una aplicación hecha de manera flexible, simple y sobretodo: funcional.

Después de haberlas probado todas, definitivamente me quedé con ésta última, Todoist. ¿Por qué? Voy a enumerarlas para facilidad:

  • Integración con mi navegador Casi todo el tiempo me la paso en Chrome y la mayoría del tiempo olvido tanto las direcciones web como las cosas por hacer (¡no por nada necesito una lista de pendientes!). Todoist me coloca un recordatorio de cuantas cosas tengo por hacer HOY y puedo marcarlas hechas, editarlas, moverlas de día, darles prioridad sin tener que ir a la web de ellos.
  • Puedo ver links Con TeuxDeux no podía ver ni siquiera la tarea completa! Tenía que colocar el cursor encima para recordar exactamente lo que tenía por hacer. Mucho consumo de tiempo valioso, además que se me hacía imposible ir a un enlace haciendole simplemente click.
  • Separación por etiquetas Es como proyectos, así puedo tener claro a qué va relacionado esa tarea, ya sea de la universidad, el trabajo o incluso simplemente tiempo libre. Esta última es de las que más he usado, ya que recuerdo en que usar mi tiempo libre de forma útil.
  • Disponible hasta para móviles Puedes entrar con un navegador móvil a todoist.com y nunca perder de vista tus pendientes.

¿Qué no me gusta? Su diseño. Aunque no es que sea MAAAALO, si le falta más innovación comparado con Strikeapp, por ejemplo.
