NetworkManager y todas las interfaces de red

September 13, 2010

Estaba buscando desde hace como una semana la razón por la que no me aparecía la opción para usar el NetworkManager para configurar mi conexión por cable de red.

Resulta que lo único que esto viene configurado para no manejarlos, sólo la WiFi. Solucionarlo es muy sencillo, se edita /etc/NetworkManager/NetworkManager.conf y se busca la sección de ifupdown debe decir algo como:

[ifupdown]
managed=false

Simplemente se cambia a true el valor managed y luego de reiniciar el demonio (/etc/init.d/network-manager restart) ya puede manejar las redes listadas en /etc/network/interfaces.
